# 🏛️ Sivas City Explorer - Class Project

## 🇺🇸 English Project Description

### 📌 About the Project
This repository contains a 1st-year Web Design class project. The main objective is to build a fully responsive web platform to promote the historical and cultural heritage of Sivas. The current version focuses on the core structural layout and visual design integrations.

### 🚀 Key Features
* **📱 Responsive Architecture:** Built with the Bootstrap 5 grid system to ensure seamless layout flexibility across desktop, tablet, and mobile screens.
* **🖼️ Modern UI Components:** Features an interactive image carousel on the homepage to highlight key locations.
* **📇 Information Cards & Modals:** The guide page presents historical sites using card layouts and utilizes modal pop-ups to display detailed descriptions and images.
* **🧭 Consistent Navigation:** Includes a sticky navigation bar and a unified footer that remain constant across all HTML pages.

### 🛠️ Technologies Used
* 🏗️ HTML5
* 🎨 CSS3
* 🧰 Bootstrap 5

## 🇹🇷 Türkçe Proje Açıklaması

### 📌 Proje Hakkında
Bu depo, 1. sınıf Web Tasarımı dersi proje ödevini içermektedir. Temel amaç, Sivas şehrinin tarihi ve kültürel miraslarını tanıtmak için mobil uyumlu bir web platformu oluşturmaktır. Mevcut sürüm, projenin temel iskelet yapısına ve görsel tasarım entegrasyonlarına odaklanmaktadır.

### 🚀 Öne Çıkan Özellikler
* **📱 Mobil Uyumlu Mimari:** Masaüstü, tablet ve telefon ekranlarında sorunsuz esneklik sağlamak için Bootstrap 5 ızgara sistemi ile tasarlanmıştır.
* **🖼️ Modern Görsel Bileşenler:** Önemli mekanları vurgulamak için anasayfada etkileşimli bir resim kaydırıcı bulundurur.
* **📇 Bilgi Kartları ve Açılır Pencereler:** Rehber sayfası, tarihi alanları kart tasarımlarıyla sunar ve detaylı açıklamaları göstermek için açılır pencerelerden faydalanır.
* **🧭 Sabit Gezinme:** Tüm sayfalarda ortak olarak çalışan ve ekranda sabit kalan bir üst menü ile standart bir alt bilgi alanı içerir.

### 🛠️ Kullanılan Teknolojiler
* 🏗️ HTML5
* 🎨 CSS3
* 🧰 Bootstrap 5
